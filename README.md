# voice_assistant
# 🎙️ AI Voice Assistant

An intelligent Python-based Voice Assistant capable of performing voice-controlled tasks such as opening applications, searching the web, answering queries, and automating basic system operations using speech recognition and text-to-speech technologies.

---

## 🚀 Features

* 🎤 Voice command recognition
* 🔊 Text-to-speech responses
* 🌐 Web searching functionality
* 📂 Open applications using voice commands
* ⏰ Time and date responses
* 🧠 AI-inspired assistant behavior
* ⚡ Fast and interactive command execution

---

## 🛠️ Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* PyAudio
* Webbrowser
* Datetime
* OS Module

---

## 📌 How It Works

1. The assistant listens to the user's voice command.
2. Speech is converted into text using Speech Recognition.
3. The command is processed and matched with predefined actions.
4. The assistant responds through voice output and executes the requested task.

---

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/14himaja/voice_assistant.git
```

Navigate to the project folder:

```bash
cd voice_assistant
```

Install required dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
